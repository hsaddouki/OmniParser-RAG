## 🚀 Características Principales
* **100% Local Inerence:** Ejecución total mediante Ollama (Llama 3.1 / DeepSeek-Coder).
* **GraphRAG Architecture:** Recuperación híbrida que combina embeddings vectoriales (ChromaDB) con relaciones semánticas en grafos (Neo4j).
* **Self-Correcting Loop:** Ciclo agéntico que valida el código generado y reintenta la búsqueda en caso de errores de sintaxis o lógica.
* **Code Topology Awareness:** Capacidad para navegar por jerarquías de clases y dependencias de funciones de forma nativa.
